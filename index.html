d3

A software deployment and patch management system for OS X, built on the Casper Suite.

Download this project as a .zip file Download this project as a tar.gz file

d3 is a software deployment and patch manangement system for OS X that enhances the capabilities of the Casper Suite from JAMF Software. It was created by Pixar Animation Studios.
